📈 SPDF Framework (Saxena Portfolio Decision Framework)

Technical analysis framework for daily execution timing (applies AFTER fundamental thesis is confirmed)

Layer 1: Trend Gate
50-day & 200-day SMAs: Above both = uptrend (favor dips) | Below both = avoid fresh buys
Layer 2: Momentum
RSI(14): <35 in uptrend = buy zone | >70 = trim/take-profit zone | MACD crossover as secondary
Layer 3: Volume (BUY only)
Signal-day volume: >1.3x 20-day average = conviction | Else wait | For TRIMS: RSI >70 suffices
Layer 4: Volatility
ATR(14): Sets stop-loss distance and position sizing | Not a signal layer

Rule: Buy fires only if Layers 1 + 2 + 3 align | All sells require re-entry price and catalyst scan
